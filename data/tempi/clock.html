For Clock Edit operations, simultaneously press PGM_A and PGM_B twice.

Channel 1

Channel 1 can be configured to emit either a 10ms trigger, or a 50% duty square wave.
Channel 1 Tap Tempo can optionally be used to set the leading tempo, disabling CH1 Human Programming.
Press Button1 to set behaviour:
LEDBehaviour
PurpleLeading Tap Tempo / 50% duty
RedLeading Tap Tempo / 10ms trigger
BlueHuman Programming / 50% duty
OffHuman Programming / 10ms trigger

Channels 2-6

Channels 2-6 can be configured to emit either a 10ms trigger, or a 50% duty square wave.
Press corresponding Button to set behaviour:
LEDBehaviour
Blue50% duty
Off10ms trigger
