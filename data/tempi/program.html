For Program Edit operations, press PGM_B twice.

Human Resolution

Human Resolution controls how strict the Human programming is quantized.
  • Press Button1 for 100% Human Resolution (most quantized)
  • Press Button2 for 50% Human Resolution
  • Press Button3 for 25% Human Resolution (least quantized)

Mod - Program Shift

When Program Shift is enabled, Sending a gate to the Mod input will move the channel settings around between the Mod enabled channels. Press Button4 to set Program Shift behaviour:
LEDBehaviour
RedPrograms shift clockwise.
BluePrograms shift counter-clockwise.
PurplePrograms shift randomly.

Mod - Run/Stop

When Run is enabled, Sending a gate to the Mod input will move the channel settings around between the Mod enabled channels. Press Button5 to set Run/Stop behaviour:
LEDBehaviour
RedRun/Stop.
BlueRun/Stop All.
PurpleAlt Run.
Button6 controls whether Run/Stop is controlled Momentarily (Red) or Toggled (Off).

Note: with Program Shift and Run/Stop enabled, then State Select is used for Program Shift and Mod is used for Run/Stop. When only one behavior is enabled, then Mod is used.
