Controls

Z
tune, or masking
X
V/oct pitch CV (fundamental)
Y
V/oct pitch CV (formant)
A
pulsar output
B
pulsar output

Parameters

pos min max default
0 0 99 0
Chooses wavetable

pos min max default
1 0 99 0
Chooses wave

pos min max default
2 0 3 0
Window, or envelope applied to pulsaret.
1: Rectangular
2: Linear attack
3: Linear decay
4: Gaussian

pos min max default
3 0 1 0
Y Mode.
When 1, X input is added to Y input, so changing fundamental frequency also changes formant frequency.

pos min max default
4 0 99 0
Masking.
When 0, Z controls tuning.
When 1-99, some pulsarets are omitted. Output B is the inverse of A - when A emits a pulsaret it is skipped on B and vice versa.
When 1, pulsarets are masked randomly.
When 2-99, pulsarets are masked in a burst with length set to the parameter, and Z controls how many of the burst are masked.

pos min max default
5 -16 8 0
Octave. Offsets X input.
