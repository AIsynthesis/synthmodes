Mode

Disastrous Peace easter egg re-uses all mode and polyphony LED states for it's own purpose. See "Handshake" below for entering/exiting instructions.

Polyphony

 One 8-note chord
Two 6-note chords
Four 3-note chords

Mode Button

Mode button controls what FX is applied to the string synth. Long-press mode for FX variant.

 Formant Filter (variant - less abrasive)

Roland-ish chorus (variant - Solina-ish chorus)
Caveman reverb (shinier variant).

Controls

FREQUENCY controls the root note of the chord.

STRUCTURE controls the chord type.

BRIGHTNESS scans through various registrations (combinations of gains for the 6 oscillators) sorted by brightness.

DAMPING controls decay time, then attack time. Drones continuously when turned fully clockwise.

POSITION controls FX amount.

IN - is fed into the FX, amount controlled by POSITION.
