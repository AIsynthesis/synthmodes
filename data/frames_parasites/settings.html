Channel Settings

Interpolation curve Hold Add for 1 second.
Channel knobs control interpolation curves per channel.
Options are step, linear, accelerating, decelerating, smooth departure/arrival, and bouncing.
Response curve Hold Del for 1 second.
Channel knobs control response curves per channel, from linear to exponential.

Save/Reset State

Frames Parasites offers a 5 save slots. Knobs 1-4 select user slots,

Save Hold Add for 3 seconds, then turn a knob to select that save slot. Turning Frame knob selects the boot settings. Press Add again to confirm save, Del to cancel.
Reset Hold Del for 3 seconds, then turn a knob to select that save slot. Turning Frame knob resets the module to blank settings. Press Del again to confirm load, Add to cancel.
