Operation

Feed steady clock into Frame CV to generate clock ticks. What happens on a given clock tick depends on Knobs 2 & 3. This mode uses programmed keyframes, but only uses the Knob 1 keyframe values.

Knob 2 controls the number of clock ticks it takes for the sequencer to step. CCW acts as clock divisor, CW randomly skips.

Knob 3 controls the number of clock ticks it takes for the outputs to shift once to thr right. CCW acts as clock divisor, CW randomly skips.

Knob 4 introduces error on the sequence values.

Fr. Step outputs a pulse when either the sequencer steps or the outputs shift.

Hold Add for 3 seconds, then use Knobs 1-4 to set pattern length. Press any button to exit.

Hold Del for 3 seconds, then use Knobs 1-4 to set envelope shape. Press any button to exit.
