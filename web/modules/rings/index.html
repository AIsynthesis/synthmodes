Modes

Modal Resonator

Mode

Solid green

Polyphony

One
Two
Four

Controls

STRUCTURE controls the frequency ratio between partials (and by doing so, the perceived structure - plate, bar, string)

BRIGHTNESS adjusts the level of higher harmonics in the signal by filtering both the exciter and damping

DAMPING controls the decay time of the sound.

POSITION controls on which point of the string/surface the excitation is applied

Mode

Polyphony

One
Two
Four

Controls

BRIGHTNESS adjusts the level of higher harmonics in the signal by filtering both the exciter and damping

POSITION controls on which point of the string/surface the excitation is applied

FM Voice

Mode

Flashing green

Polyphony

One
Two
Four

Controls

STRUCTURE controls the frequency ratio.

BRIGHTNESS adjusts the FM index.

DAMPING controls the FM index and amplitude decay.

POSITION Feedback path (no feedback at 12 o'clock)

IN goes into an envelope follower changing FM index and output amplitude.

String & Reverb

Mode

Flashing red

Polyphony

One
Two
Four

Controls

STRUCTURE controls the amount of modulation and detuning of the partials.

BRIGHTNESS adjusts the level of higher harmonics in the signal by filtering both the exciter and damping. Also controls reverb lowpass filter.

DAMPING controls the decay time of the sound, as well as reverb amount and time.

POSITION controls on which point of the string/surface the excitation is applied



Disastrous Peace

Mode

Disastrous Peace easter egg re-uses all mode and polyphony LED states for it's own purpose. See "Handshake" below for entering/exiting instructions.

Polyphony

One 8-note chord
Two 6-note chords
Four 3-note chords

Mode Button

Mode button controls what FX is applied to the string synth. Long-press mode for FX variant.

Formant Filter (variant - less abrasive)

Roland-ish chorus (variant - Solina-ish chorus)
Caveman reverb (shinier variant).

Controls

FREQUENCY controls the root note of the chord.

STRUCTURE controls the chord type.

BRIGHTNESS scans through various registrations (combinations of gains for the 6 oscillators) sorted by brightness.

DAMPING controls decay time, then attack time. Drones continuously when turned fully clockwise.

POSITION controls FX amount.

IN - is fed into the FX, amount controlled by POSITION.
